# 🌈 五年级英语学习乐园

一个专为五年级学生设计的趣味英语学习网站，帮助孩子轻松掌握核心词汇和句型。

## 📚 功能特点

- **6个单元内容**：涵盖健康饮食、绿色生活、友谊、自我成长、未来规划等主题
- **72个核心词汇** + **36个句型** + **12个情景对话**
- **7种学习模式**：
  - 📖 单词卡片 - 翻卡学习核心词汇
  - 🖼️ 看图选词 - 看图片选择正确单词
  - ✍️ 拼写练习 - 看中文拼写英文单词
  - 🔀 句子排序 - 将打乱的单词排成句子（含配对变体）
  - 📝 填空选择 - 选词填空完成句子
  - 🎯 情景对话 - 选择正确的回应
  - ⏱️ 限时挑战 - 60秒内答对更多题目

## 🚀 快速开始

### 方法一：直接打开
双击 `learning-website/index.html` 在浏览器中打开

### 方法二：本地服务器
```bash
cd learning-website
python -m http.server 8080
```
然后访问 http://localhost:8080

## 📁 目录结构

```

├── learning-website/    # 网站主目录

│   ├── style.css        # 样式表
```

## ✨ 技术栈

- 纯 HTML + CSS + JavaScript
- 无需任何框架或构建工具
- 使用 Web Speech API 实现单词发音
- 响应式设计，支持电脑和平板
