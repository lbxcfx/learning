# 五年级英语学习乐园 - 微信小程序

## 📱 项目简介

五年级英语学习小程序，帮助学生轻松掌握核心词汇和句型。

## ✨ 功能特点

- 📖 **单词卡片** - 翻卡学习核心词汇
- 🖼️ **看图选词** - 看图片选择正确单词
- ✍️ **拼写练习** - 看中文拼写英文单词
- 🔀 **句子排序** - 将打乱的单词排成句子
- 🏆 **排行榜** - 查看学习排名
- 👤 **微信登录** - 保存学习进度

## 🚀 使用方法

### 1. 安装微信开发者工具

下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `learning-mini` 文件夹
4. 填入您的 AppID（在 project.config.json 中也需要修改）

### 3. 开发调试

点击"编译"预览小程序

### 4. 发布上线

1. 点击"上传"将代码上传到微信后台
2. 在微信公众平台提交审核
3. 审核通过后发布

## 📁 项目结构

```
learning-mini/
├── app.js              # 小程序入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
├── sitemap.json        # 站点地图
├── pages/
│   ├── index/          # 首页（单元选择、模式选择）
│   ├── learn/          # 学习页（各种学习模式）
│   ├── rank/           # 排行榜页
│   └── profile/        # 个人中心页
├── utils/
│   └── data.js         # 学习数据
└── images/             # 图片资源（需添加tab图标）
```

## ⚠️ 注意事项

1. 请在 `project.config.json` 中替换 `appid` 为您自己的小程序 AppID
2. 底部 tabBar 图标需要自行添加到 `images/` 目录
3. 排行榜功能需要开通云开发后接入真实数据
